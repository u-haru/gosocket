# websocks

```Socks5 over Websocket```

This is experimental.

Usage of server:  
  -h string  
        Listening Address:Port/Path (default "0.0.0.0:8080/ws")  

Usage of client:  
  -h string  
        Listening Address:Port (default "0.0.0.0:8000")  
  -t string  
        Target websocket path (default "ws://127.0.0.1:8080/ws")
